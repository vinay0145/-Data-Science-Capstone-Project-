# E-Commerce Customer Analysis

## Objective
To analyze customer behavior and identify high-value customers.

## Dataset
Dataset taken from Kaggle.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Steps Performed
1. Data Loading
2. Data Cleaning
3. Feature Scaling using StandardScaler
4. Clustering (KMeans)
5. Data Visualization

## Key Insights
- High income customers spend more
- Some customers have low engagement
- Clustering helped identify target groups

## Conclusion
This project helps businesses understand customer segments and improve marketing strategies.
